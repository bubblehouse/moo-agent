# Mason Learned Rules

## Lessons Learned

- `@tunnel` must be on its own SCRIPT: line — never combine with DONE: in the same response. DONE: after a SCRIPT: block is correct; DONE: on the same line as the command is not. Wrong: `@tunnel south to #19 done.` — the "done." becomes part of the command and errors. Correct: put `@tunnel south to #N` alone in SCRIPT:, then DONE: as a separate line.
